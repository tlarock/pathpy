Which development tools do you recommend?

While you are obviously free to use the python editor of your choice, we recommend new users to have a look at the development environment Visual Studio Code, a platform-independent Open Source code editor available for Windows, MacOS, and Linux. Just download the installation file and run the setup. Once the installation has completed, run Visual Studio Code either by clicking the icon or by typing code in the terminal.

A major benefit of Visual Studio Code is its built-in support for jupyter notebooks, which will allow you to edit python code in a convenient editor, while seeing the interactive output of your code in a split window. pathpy provides dedicated support for embedded visualisations, so you can display interactive visualisations of networks, temporal networks, and higher-order networks directly in Visual Studio Code. The following video demonstrates this:

To conveniently work with python and jupyter in Visual Studio Code, you will need to install the python extension, which adds python code editing, debugging, and linting functionality as well as jupyter support. You can install it free of charge directly from Visual Studio Code's integrated extension manager. Open Visual Studio Code and click the module icon in the bottom of the left menu bar or press Ctrl+Shift+X. This will bring up the Extensions window. Type python and click the first result. In the window on the right, click install. When the download is complete, click Reload to complete the installation.

Now that we have installed all necessary tools and packages, let us verify that our environment is set up properly. Create a new file in Visual Studio Code and add the following code: 

#%%
import pathpy as pp
n = pp.Network()
n.add_edge('a', 'b')
n.add_edge('b', 'c')
print(n)

#%%
n
Once you have typed the code, you will see that two so-called code lenses Run cell will appear above the #%% tags. These tags mark the start of a cell in a jupyter notebook that we can execute directly from within Visual Studio Code. If you click the code lens, a new jupyter kernel will be started automatically, and the code in the current cell will be executed. An output window will open that displays the output of your code.

This gives you a very convenient way to complete data science tasks for (time series) data on complex networks. To learn more about how you can use pathpy to read, analyze, and visualize such data, just move on to the next unit.
